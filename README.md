# SASTCsharpFriendPart

这里有一群可爱的动物小伙伴~
目前这个项目还是仅仅只有最基本的登录/注册界面，我们希望你能够为当前的桌面端 APP 添加供小动物们沟通交流的交流界面。
注意：对于本套题目，我们最注重的部分正是 **消息传输框架** 的实现

## （1）基础要求

- [ ] 在自己的电脑上完成数据绑定
- [ ] 实现或套用一套消息传输框架
- [ ] 为当前应用搭建出一套聊天交流界面（参考 Telegram、QQ、WeChat 等均可）

## （2）进阶要求

- [ ] 为消息传输框架添加一套单元测试
- [ ] 把数据库部署到云端（neon、Azure DB、AWS、阿里云、腾讯云甚至华为云均可）
- [ ] 部署这个社交软件（准确来说应该是部署他的底层服务

## （3）开放内容

1. 添加语音聊天功能！
2. 顺手尽可能美化一下你的前端
3. 添加其他你觉得可能用得到的功能
